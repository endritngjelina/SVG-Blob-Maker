# 🫧 SVG Blob Maker

A lightweight, browser-based tool for generating organic, customizable SVG blob shapes. Built entirely with vanilla HTML, CSS, and JavaScript — no frameworks or dependencies required.

---

## 🚀 Features

- Generate smooth, organic blob shapes instantly
- Customize blob complexity and variance
- Copy SVG code directly to your clipboard
- Download blobs as `.svg` files
- Multiple iterations and design explorations (v0–v13)
- Works fully in-browser — no installation needed

---

## 🛠️ Usage

1. Clone or download the repository:
   ```bash
   git clone https://github.com/endritngjelina/SVG-Blob-Maker.git
   ```
2. Open any `.html` file in your browser — no server required.
3. Recommended: Open **SVG Blob Maker 13.html** for the most recent version.
